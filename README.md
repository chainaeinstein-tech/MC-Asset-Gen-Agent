# Minecraft Asset-to-Code Pipeline Agent

## 🚀 项目简介
这是一个专为《我的世界》网易版（Python 2.7）模组开发设计的自动化资产生成代理。它通过 AI 逻辑驱动，实现了从“自然语言需求”到“游戏内可用代码/配置”的全链路自动化。

## 🛠 核心功能
- **意图解析 Agent**: 能够自动分析玩家对物品、方块的描述，并拆解为具体的技术参数。
- **自动化配置生成**: 基于预设模板自动生成符合 MC 规范的 JSON 配置文件。
- **SDK 适配层**: 针对 Python 2.7 环境优化，确保生成的代码可直接在网易版 Mod 环境中运行。

## 📂 项目结构
- `logic_agent_core.py`: 核心逻辑链推理模块。
- `resource_generator_agent.py`: 资源模板填充 Agent。
- `mod_sdk_adapter.py`: 网易 Mod API 适配层。

## 📈 影响力与价值
该工具将原本繁琐的物品注册流程从数小时缩短至分钟级，极大地提升了开发者在原型阶段的迭代效率。
